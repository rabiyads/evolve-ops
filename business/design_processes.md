| **Design Field** | **Phase** | **Goal** | **Deliverables** |
|------------------|-----------|-----------|------------------|
| **UX / UI Design** | 1- Research & Discovery | Understand users, business goals, and define the problem. | User personas, empathy maps, research summary, problem statements |
|  | 2- Strategy & Planning | Define the direction, structure, and flow of the product. | Information architecture, user flows, site map, product brief |
|  | 3- Wireframing | Establish layout, hierarchy, and navigation before visuals. | Low-fidelity wireframes, diagrams, workflows |
|  | 4- Ui design | Create the visual language and final screen designs. | High-fidelity screens, style guide, design system |
|  | 5- Prototyping | Check usability and make design functional to check user experience | Fully functional prototype|
|  |6- Developer Handoff | Ensure design is ready for implementation. | Figma handoff file, assets, design specifications |
| **Branding Design** | 1- Research & Strategy | Understand brand values, market, and audience. | Brand audit, competitor analysis, moodboard |
| | 2- Concept Development | Create identity concepts aligned with brand goals. | Logo sketches, concept boards |
| | 3- Brand Guidelines | Document the brand usage rules. | Brandbook |
| **Print Design** | 1- Research & Brief | Understand objectives, content, and format. | Project brief, creative direction |
|  | 2- Visuals  | Arrange elements  | Photography or illustrations or calligraphy |
|  | 3- Design & Review | Place design in mockups | Final design mockups |
|  | 4- Prepress & Delivery | Prepare files for production. | Print-ready PDFs, production specs |
